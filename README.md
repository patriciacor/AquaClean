# AquaClean

App para mantenimiento de piscinas.

## Setup

1. Flutter SDK >= 3.0.0
2. Firebase configurado para Android e iOS
3. Configurar variables en Codemagic para keystore Android
4. Ejecutar `flutter pub get`
5. Construir con `flutter run`
